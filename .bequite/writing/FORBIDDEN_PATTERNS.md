# Forbidden Patterns

> Patterns the output must NEVER contain. Output is grepped against this list before delivery (Writing DNA quality gate). Two layers: defaults (every profile) + profile-specific.

## Default bans (generic-AI tells — always active)

- "delve" · "tapestry" · metaphorical "landscape"/"navigate" · "harness the power"
- "in today's fast-paced world" · "in an ever-evolving" · "it's important to note that"
- furthermore/moreover/additionally chains (≥2 in one paragraph)
- Uniform sentence lengths (metronome rhythm)
- Triple-adjective stacks · hedging stacks ("could potentially perhaps")
- Every-section-ends-with-summary structure
- Rhetorical-question openers used more than once per piece
- Exclamation marks in formal registers
- Em-dash density above ~1 per 80 words
- Listicle structure where prose was requested
