## Notes

- A hidden gem is **not** a guarantee of payout. Every entry has trust check + risk level.
- Re-evaluated on repeat searches; entries can be marked: 🆕 new / ✅ still active / ❌ expired / ⚠ risk increased / ⬆ better alternative / 🔍 needs manual verification.
- The agent searches in multiple languages when relevant (English / Arabic / Spanish / Portuguese / German / French / Italian / Turkish / Polish / Lithuanian / Romanian / Indonesian / Hindi).
