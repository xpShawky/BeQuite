# Starting Path Advisor — `/bq-start` (C17) — alpha.24

Help a user choose **how to start** — most people fail at starting/choosing, not building. Command: `bq-start.md`. Skills: product-strategist + make-money + job-finder + writing-dna + researcher + localization-rtl.

## Why a command

Distinct advisory workflow with a 12-artifact decision set; integrates with brand/course/offer/career commands as the upstream "which path" decision. Neutral and concrete — not motivational filler.

## Outputs — `.bequite/start/`

START_CONTEXT · OPTION_MAP · AUDIENCE_AND_PLATFORM_ANALYSIS · ACCOUNT_STRATEGY · CONTENT_POSITIONING · PERSONAL_BRAND_VS_BRAND_DECISION · PLATFORM_PROFILE_SETUP · FIRST_30_DAYS_PLAN · EXPERIMENT_PLAN · RISKS_AND_TRADEOFFS · FINAL_RECOMMENDATION · NEXT_STEPS.

## Decision frameworks

1. **Personal profile vs page** — existing reach vs relevance, unfollow/social risk, algorithmic reach, page-from-zero problem; often: warm-start validation on the profile → dedicated page when the promise is clear → both with distinct roles.
2. **Personal name vs brand** — trust/flexibility/pivoting/searchability/monetization; is the user the product? should it survive beyond them?
3. **Platform strategy** — per relevant platform (FB profile/page/group · X · LinkedIn · YouTube · TikTok · Instagram · GitHub · website · newsletter): best use, setup, format, risks, first action.
4. **Niche decision** — one niche / umbrella + pillars / separate accounts / phased — via audience-overlap + monetization + confusion-risk + experimentation period (not blind narrow-niche, not random mixing).
5. **Course/content/offer sequence** — content-first / free mini-course / cohort / paid / YouTube series / lead magnet / offer-first / community-first → integrates C5/C11/C6/C2/C1/C15/C14.
6. **Career/freelance start** — LinkedIn positioning / CV / GitHub-portfolio / job-finder / proposal / content authority / proof projects → integrates C9/C8/C10/C11/C14.

## Output + hard rules

FINAL_RECOMMENDATION = recommended option · why · what NOT to do · 7-day start · 30-day experiment · profile setup checklist · content pillars · first 10 post/video ideas · metrics (saves/DMs/comments, not just likes) · pivot trigger · next commands. Concrete decisions, not "be consistent/post value/know your audience." Neutral (disagree when evidence says so); use the user's actual assets; respect privacy/audience-mismatch; always end with a recommendation + action. **Built alpha.24 — NOT live-tested.**
