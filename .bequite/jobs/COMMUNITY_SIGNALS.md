## Methodology

Captured by Claude when researching with available tools (WebFetch / WebSearch / Chrome MCP / Computer Use MCP). Public conversations only — no auth-walled or private content.

Per BeQuite safety rules:
- ❌ Don't capture scammy claims; mark + skip
- ❌ Don't capture VPN-misrepresentation tutorials
- ❌ Don't capture CAPTCHA-bypass discussions
- ✅ Capture legitimate payout proof + community-verified opportunities
- ✅ Mark uncertainty clearly
- ✅ Cite source URL + date so user can verify
