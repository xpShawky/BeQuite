## How to use this file

When you run `/bq-job-finder AI_assisted=true`:

1. Claude reads `JOB_PROFILE.md` to know which AI tools you have
2. Cross-references the categories above
3. Surfaces opportunities where your AI tool stack gives you leverage
4. Writes opportunities here + main entries in `OPPORTUNITIES.md`
5. For each AI-assisted opportunity, includes the specific AI tool that gives you an edge + how to demonstrate fluency in the application

The goal: find work where being AI-fluent isn't a "nice to have" — it's the **multiplier** that makes the work profitable for you.
