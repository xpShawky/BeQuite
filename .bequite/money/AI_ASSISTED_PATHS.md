## How to use this file

When `/bq-make-money AI_assisted=true` runs:

1. Claude reads `MONEY_PROFILE.md` to know your AI tool stack + country + payout methods + risk tolerance + skills
2. Cross-references the categories above
3. Surfaces paths where your AI stack is the **multiplier** — not just a tool, but the reason the work is profitable for you
4. Writes paths here + main entries in `OPPORTUNITIES.md`
5. Includes specific demo / portfolio items per category so you can hit the ground running

The goal: don't just find "make money tasks". Find paths where you can outperform non-AI workers because your AI stack does the heavy lifting in 1/10th the time.

---

## Notes on tool neutrality

The AI tools listed (Claude, ChatGPT, Midjourney, Runway, n8n, etc.) are **candidates**. The user's actual AI stack — whatever they've already paid for and learned — drives the recommendation.

The agent doesn't push tools the user doesn't have. It finds work that matches the tools they DO have.

See `.bequite/principles/TOOL_NEUTRALITY.md`.
