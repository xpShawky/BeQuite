# Screenshots

Visual QA + Design Continuity + live-edit screenshots land here (when browser automation — Playwright MCP or project Playwright — is available). Naming:

- `before-<ts>-<section>.png` / `after-<ts>-<section>.png` — live edits
- `continuity-<route>-<section>-<ts>.png` — Design Continuity Gate
- `qa-<route>-<viewport>-<ts>.png` — Visual QA pass
- `v<N>-{desktop,mobile}.png` — UI variants

If no browser automation is available, this stays empty and reports note "code inspection + manual screenshots requested." BeQuite never auto-installs Playwright to populate this folder.
