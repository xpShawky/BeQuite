# Pitch Templates

Reusable application / outreach templates per opportunity type. User fills in name + portfolio links + role-specific details.

Each template ≤ 200 words. Saved by `/bq-job-finder` after every search.

---

## AI gig task application

```
Hi <platform / hirer>,

I'd like to take this task. I work with <AI tools listed>; for tasks like
<task type> I typically <approach>. Sample of my output: <portfolio link>.

I can deliver <quantity / quality> within <timeframe>. Available <hours
per week> hours per week, <timezone>.

Payout via <preferred method>. Available now.

— <name>
```

## Freelance proposal (Upwork-style)

```
Hi <client>,

I noticed you need <specific deliverable from job post>. I've built <similar
deliverable> for <past context>; you can see it at <portfolio link>.

For your project, I'd <concrete first step + timeframe>. My rate for this
scope is <amount>. I can start <when>.

Two questions:
1. <relevant clarifying question 1>
2. <relevant clarifying question 2>

— <name>
```

## Remote full-time application

```
Hi <hiring team>,

I'm applying for <role>. Three sentences on fit:

1. I've done <relevant achievement> with <relevant tool/skill>.
2. I work async-friendly in <timezone>, comfortable with <work style>.
3. <One concrete thing I'd bring> — e.g. "I cut deploy time from 12 min to 2 min at <past role>".

Portfolio: <link>
Availability: <when>
Salary expectation: <range>

— <name>
```

## LinkedIn cold outreach

```
<Hi / Hello> <name>,

Saw your post about <specific topic>. The part about <specific detail>
caught my attention because <one-sentence relevant context>.

I work with <AI tools / skills> and <one-line credibility>. If you ever
need help with <specific service>, happy to send <pitch / sample>. No
worries if not — just wanted to connect.

— <name>
```

## Local-language application (Egypt / Arabic example)

```
السلام عليكم،

شفت إعلانكم عن <role>. أنا <name> من <city>.

عندي خبرة في <skill> ومستخدم <AI tools>. مثال على شغلي: <portfolio link>.

ممكن أبدأ <when> ومتاح <hours> ساعات أسبوعياً. السعر <amount> أو حسب
السكوب.

شكراً
— <name>
```

(Localize for other languages as needed: Spanish / Portuguese / French / German / Turkish / Hindi / etc.)

---

## Tips per template

- **AI gig:** lead with tool fluency + sample output
- **Freelance:** propose a concrete first step, not "I'm interested"
- **Remote full-time:** 3 sentences max; portfolio link does the heavy lifting
- **LinkedIn cold:** relevance reason + soft ask + opt-out
- **Local language:** match formality + cultural conventions

Iterate based on what gets responses (track in `APPLICATION_TRACKER.md`).
