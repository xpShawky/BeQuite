## Notes

- A "hidden gem" doesn't mean guaranteed payout. Every entry has a trust check + risk level + confidence level.
- Re-evaluated on repeat searches; entries are marked: 🆕 new / ✅ still active / ❌ expired / ⚠ risk increased / ⬆ better alternative / 🔍 needs manual verification.
- Multi-language search when relevant: English / Arabic / Spanish / Portuguese / German / French / Italian / Turkish / Polish / Lithuanian / Romanian / Indonesian / Hindi.
- Tool neutrality applies: named platforms are candidates, not endorsements.

## Common categories of hidden gems

Categories Claude searches for in this mode (illustrative, not exhaustive):

- **Country-specific microtask platforms** (Yandex Toloka, Wuzzuf, Brighter Monday, Get on Board, Wantedly, etc.)
- **AI training task platforms** (Outlier, Mercor, Mindrift, Surge, Data Annotation, Scale, Appen, Labelbox)
- **Research panels** (User Interviews, Respondent, dscout, Prolific, Wynter)
- **Testing platforms** (UserTesting, UserBrain, Userlytics, Lookback, PlaybookUX, TryMyUI)
- **App-based earning programs** (Premise, Field Agent, Streetbees — legitimate location-based)
- **Niche freelance markets** (regional boards in user's country / language)
- **Small companies hiring globally** (Wellfound, RemoteRocketship)
- **Newly-launched creator tools paying testers** (caught via community signals)
- **Translation / localization niches** (Gengo, ProZ, OneHourTranslation, Unbabel + regional)
- **AI tool reviewer programs** (newly launched AI products paying real users for feedback)

The agent doesn't limit to these — it searches live and surfaces what fits the user's profile, country, languages, AI tool stack, and risk tolerance.
